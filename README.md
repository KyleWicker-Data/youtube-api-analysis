# YouTube API Analysis

This project analyzes YouTube video comment data using the YouTube Data API.
The goal is to explore engagement patterns and trends in comments on a popular Minecraft trailer.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- YouTube Data API

## Files
- `MINCRAF_MOVIESortingPlottingData.ipynb`: Data collection and analysis
- `final_minecraft_comments.csv`: Extracted comment data
- `Documentation.docx`: Project documentation

## Notes
API keys are handled securely using environment variables and are not included in this repository.
